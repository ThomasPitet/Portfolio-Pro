The League

Application web de gestion d'une ligue e-sport -- equipes, joueurs, matchs et performances.

Presentation du projet

The League est une application web PHP developpee en architecture MVC. Elle permet de consulter et gerer une ligue de jeu video competitif : equipes, joueurs avec leurs statistiques, matchs passes et performances individuelles.

Le projet a ete realise en equipe dans le cadre de la formation, avec une repartition claire des taches. Il met en pratique les concepts PHP avances : routeur personnalise, managers de donnees, modeles orientes objet et templates PHTML.

Architecture MVC

Le projet suit une architecture Model-View-Controller rigoureuse :

Models

Team, Player, Game, PlayerPerformance -- classes PHP representant les entites BDD.

Managers

Couche d'acces aux donnees : requetes SQL encapsulees dans des classes dediees.

Controllers

PageController gere chaque route : recupere les donnees et les passe aux templates PHTML.

Router

Routeur custom qui dispatch les requetes GET vers les bonnes methodes du controleur.
